# Linkedu

Plataforma que conecta estudantes à oportunidades acadêmicas.

## Dependências necessárias

- node 22.18.0
- npm 10.9.3

## Inicializando a aplicação

Rode o seguinte comando para instalar as dependências necessárias:

```bash
npm install
```

Após isso, execute a aplicação:

```bash
 npm start
```

O servidor estará rodando na porta 3000. Para acessar, digite no navegador:

```
http://localhost:3000
```
